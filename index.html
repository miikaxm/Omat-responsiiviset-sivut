Taidot

Python

Olen suorittanut MOOCin Python-kurssin ja tehnyt ryhmätyönä Hirsipuu-pelin Pythonilla. Tämä on antanut minulle hyvän perustan ohjelmoinnin perusteista ja Pythonin käytöstä.

Html

Olen tehnyt useita verkkosivuja käyttäen HTML-kieltä, kuten omat etusivut ja responsiiviset sivut. Tämä on antanut minulle hyvän ymmärryksen HTML:n käytöstä verkkosivujen rakenteen luomiseen.

Css

Olen käynyt läpi Sololearnin CSS-tutoriaalia ja oppinut CSS:n perusteet. Olen myös käyttänyt CSS:ää omien verkkosivujeni tyylittelyyn, mikä on auttanut minua ymmärtämään, miten luoda visuaalisesti miellyttäviä ja responsiivisia sivustoja.

Bootstrap 5

Olen käyttänyt Bootstrap 5:sta kehittäessäni responsiivisia verkkosivuja. Lisäksi olen oppinut Bootstrapin käyttöä katsomalla opetusvideoita ja tutustumalla sen dokumentaatioon, mikä on auttanut minua ymmärtämään sen tarjoamia komponentteja ja työkaluja.

Git

Olen oppinut Gitin perusasiat, kuten versionhallinnan perusteet, haarojen käytön ja muutosten hallinnan. Lisäksi olen käyttänyt Git Bashia projektien hallintaan ja versionhallinnan tehtäviin.

Github

Olen käyttänyt Githubia projektieni hallintaan ja versionhallintaan. Lisäksi olen osallistunut ryhmätyöhön, jossa käytimme Githubia yhteistyön koordinointiin. Olen opetellut Githubin peruskäytön, kuten repositorioiden luomisen, haarojen hallinnan ja muutosten yhdistämisen.
